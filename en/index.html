In brief
ToutDoux is a project manager for GNOME.
The approach to data manipulation is abstract (database free).
It's extensible with plug-ins.
As much as possible, tasks are and will be subcontracted to front-ends (command line).
The file format is XML (including structure, plug-ins parameters and datatables).
