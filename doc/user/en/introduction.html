Chapter 1. Introduction

ToutDoux is a project manager for GNOME.

  • The approach to data manipulation is abstract (database free).

  • It's extensible with plug-ins.

  • As much as possible, tasks are performed by (and will be assigned to) front-ends (command line).

  • The file format is XML (this includes dababase structure, information related to plug-ins and datatables).

Phonetically "ToutDoux" and "ToDo" are equivalent. "ToutDoux" - or rather "tout doux" - is a French expression. It means "all sweet" or "all's well", "going smoothly", "it's cool".
