Unix distributions's specificities

GNU/Linux Red Hat 6.2 : no known issues.

GNU/Linux Mandrake 7.1 : Mandrake 7.1 includes the libreadline.4.1 library. But PostgreSQL 7.0.2 only gets along with version 3 ! As a result, the package (PostgreSQL) must be installed with the option --nodeps. Also, create a link called libreadline.3 and pointed at libreadline.4.1. (thanks to Sébastien DINOT <sdinot@april.org>)
