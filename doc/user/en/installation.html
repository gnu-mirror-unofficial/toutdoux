Requirements

These are the elements (libraries, software and scripts) needed for using ToutDoux.

Based on GNOME, the requirements are :

  • glib-1.2.1 : C library

  • gtk+-1.2.1 : toolkit

  • gdk-pixbuf-0.9.0 : image file manipulation

  • gnome-libs-1.0.8 : environment

  • libxml-2.3.5 : XML file manipulation

  • gettext-0.10.35 : internationalization

  • getxml-1.0.3 : XML file internationalization

  • perl-5.6.0 : interpreter

  • sed-3.02 : text processor

  • jade-1.2.1 : text processor

  • stylesheets-0.13-4 : collection of stylesheets (SGML/Docbook).
