Compilación

Las opciones de configuración son :

  • --enable-gtk-doc=ARG : utiliza gtk-doc para construir la documentación [por defecto=auto, no]

  • --enable-lang-en=ARG : añade a la documentación en inglés [ por defecto=yes, no]

  • --enable-lang-fr=ARG : añade a la documentación en francés [ por defecto =auto, yes, no]

El procedimiento es famoso :

  • $ ./configure : configuración

  • $ make : compilación

  • $ su : entrada como superusuario

  • $ make install : instalación

  • $ exit : salida de superusuario.
