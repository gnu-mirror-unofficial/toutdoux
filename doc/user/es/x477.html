Peculiariedades de las distribuciones Unix

GNU/Linux Red Hat 6.2 : no se han notificado particulariedades.

GNU/Linux Mandrake 7.1 : The Mandrake 7.1 proporciona la biblioteca libreadline.4.1. Pero PostgreSQL 7.0.2 sólo necesita la 3 ! Así, necesitamos instalar el paquete (PostgreSQL) con --nodeps y crear un enlace llamado libreadline.3 que apunte a libreadline.4.1. (gracias a Sébastien DINOT <sdinot@april.org>)
