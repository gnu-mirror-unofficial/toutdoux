Necesidades

Las necesidades con los elementos (bibliotecas, software y scripts) utilizados por ToutDoux.

Basado en GNOME, las necesidades son :

  • glib-1.2.1 : biblioteca C

  • gtk+-1.2.1 : toolkit

  • gdk-pixbuf-0.9.0 : manipulación de ficheros de imágenes

  • gnome-libs-1.0.8 : entorno

  • libxml-2.3.5 : manipulación de ficheros XML

  • gettext-0.10.35 : internationalización

  • getxml-1.0.3 : internationalización de ficheros XML

  • sed-3.02 : procesadores de textos

  • jade-1.2.1 : procesadores de textos

  • stylesheets-0.13-4 : colección de hojas de estilo (SGML/Docbook).
