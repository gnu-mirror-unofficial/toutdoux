La compilation

Les options de configuration sont :

  • --enable-gtk-doc=ARG : utilise gtk-doc pour générer la documentation [par défault=auto, no]

  • --enable-lang-en=ARG : ajoute l'Anglais à la documentation [par défault=yes, no]

  • --enable-lang-fr=ARG : ajoute le Français à la documentation [default=auto, yes, no]

La procédure de compilation reste un grand classique :

  • $ ./configure : configuration

  • $ make : compilation

  • $ su : connexion en administrateur

  • $ make install : installation

  • $ exit : déconnexion de l'administrateur.
